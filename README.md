
## Steam Top 100

这是一个展示 Steam 好评如潮 TOP100 游戏的前端项目，支持分页浏览、排序、价格展示等功能。

## 主要功能
- 展示游戏封面、标题、作者、类型、星级、好评数、当前价格、历史最低价
- 支持分页浏览，每页10个游戏
- 支持按排名和价格升序/降序排序
- 响应式简洁UI，适合桌面和移动端


## 技术栈
- React 19
- TypeScript
- Tailwind CSS
